# EBAM_G-Code

Создание приложения для расчета параметров 3D наплавки.

## Текущая рабочая версия

Актуальная рабочая версия: **v4.2 SPEED MM/MIN AUDITED**.

Скорость движения F в интерфейсе и G-code задаётся в **мм/мин**. Подача проволоки задаётся в **мм/с**.

## Безопасность

Приложение выдаёт расчётные стартовые режимы. Перед реальной наплавкой обязательны preview, audit, сухой прогон без луча и проволоки, проверка W-ретракта и короткий TEST Z10-15 мм.
